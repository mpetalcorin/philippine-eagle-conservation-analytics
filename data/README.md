# Data provenance and safeguards

The repository contains literature benchmarks and fully synthetic records generated by the notebook.

- `benchmarks/` contains numerical values transcribed from the cited peer-reviewed literature.
- `synthetic/` contains simulated territories, breeding histories, genomic summaries, haplotypes, and pairing plans.

No real nest coordinates, telemetry locations, bird identities, or individual genotypes are included. The synthetic files are for reproducible method development, education, and scenario analysis. They must not be interpreted as field observations or used to infer sensitive locations.
